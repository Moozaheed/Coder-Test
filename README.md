# Coder-Test

A minimal PHP, JavaScript, HTML, and CSS project configured for Dev Containers.

## Running the project

1.  Open this project in a Dev Container (using VS Code or `@devcontainers/cli`).
2.  The PHP server will start automatically on port 80.
3.  Visit `http://localhost:80` (or the forwarded port) to see the application.

## Project Structure

- `public/`: Web-accessible files (PHP, CSS, JS).
- `Dockerfile`: Configuration for the PHP Apache server.
- `.devcontainer/`: Dev Container configuration.
